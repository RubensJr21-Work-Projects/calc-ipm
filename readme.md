# Usando projeto

## Criando Ambiente virtual

```powershell
python -m venv .venv
```

## Ativar o .venv

```powershell
source .venv\bin\activate
```

## Instalando dependências

```powershell
pip install -r <versão>\requirements.txt
```

## Rodar script de processamento

```powershell
python <versão>\excel_to_csv.py
```

## Desativar o .venv

```powershell
deactivate
```
